Welcome to angular-http-interceptor and angular-spa examples

This page is public, but the Profile page needs a login. The login use Keycloak server for its authentication in the port 8090

The fastest way to get keycloak up and running is using docker.
Run this command in terminal:
docker run --name keycloak -p 9080:8080 -e KEYCLOAK_USER=admin -e KEYCLOAK_PASSWORD=admin -d jboss/keycloak

You need to create a public client for this application. Check app.module.ts for the parameters
